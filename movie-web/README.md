## Usage

### Project setup

#### Install yarn first.
```
npm install yarn
```
#### Use yarn to add all packages
```
yarn
```

#### Compiles and hot-reloads for development
```
yarn dev
```

#### Compiles and minifies for production
```
yarn build
```
